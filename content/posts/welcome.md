ASKUARY 是一座**宇宙门户**——粒子地球、深空流星、光点记忆，以及可以下滑探向的黑洞。

## 为什么重建

旧站 askuary.cn 上的体验很好，但 WordPress 与主题耦合较深。这个仓库把视觉与交互延续下来，用 Vite + TypeScript + Canvas 独立重写，方便迭代博客与部署。

## 接下来

- 在 `content/posts/` 里新增 Markdown 即可
- 运行 `npm run dev` 或 `npm run build` 会自动生成文章页
- 首页「博客」恒星与黑洞穿越均指向 `/blog/`

欢迎在这片宇宙里记录思考。
